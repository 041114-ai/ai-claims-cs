# 医疗险理赔流程

## 一、理赔前准备

### 确认保障范围
- 查看保单确认医疗险类型（门诊/住院/重疾）
- 确认就诊医院是否在保障范围内
- 了解免赔额和赔付比例

### 就诊注意事项
- 选择定点医院或二级以上公立医院
- 使用本人实名就诊
- 保留所有原始票据和资料

## 二、理赔申请流程

### 第一步：准备材料

#### 基础材料
1. 理赔申请书
2. 被保险人身份证复印件
3. 银行卡复印件
4. 保单复印件

#### 医疗材料
1. 门诊病历或住院病历
2. 医疗费用发票原件
3. 费用明细清单
4. 检查报告单
5. 诊断证明书
6. 出院小结（住院）

#### 特殊材料
- 意外事故证明（意外医疗）
- 医保结算单（有医保报销）
- 转诊证明（转院治疗）

### 第二步：提交申请

#### 提交方式
1. **线上申请**：通过保险公司APP或小程序
2. **线下提交**：到保险公司柜面
3. **邮寄申请**：邮寄至理赔中心

#### 线上申请流程
1. 登录保险公司APP
2. 选择"理赔申请"
3. 填写理赔信息
4. 上传材料照片
5. 提交审核

### 第三步：审核处理

#### 审核内容
- 材料完整性和真实性
- 事故是否符合保险责任
- 费用是否合理
- 是否存在免责情形

#### 审核时效
- 简单案件：1-3个工作日
- 一般案件：3-5个工作日
- 复杂案件：5-10个工作日

### 第四步：理赔支付

#### 支付方式
- 银行转账（主要方式）
- 支付宝/微信（部分公司支持）

#### 到账时间
- 审核通过后1-3个工作日

## 三、理赔金额计算

### 计算公式
```
理赔金额 = (合理医疗费用 - 免赔额 - 已报销金额) × 赔付比例
```

### 示例
- 医疗费用：10000元
- 免赔额：1000元
- 医保报销：4000元
- 赔付比例：100%

理赔金额 = (10000 - 1000 - 4000) × 100% = 5000元

## 四、常见问题

### Q1: 门诊和住院理赔有什么区别？
A: 门诊理赔材料相对简单，住院理赔需要更多病历材料。

### Q2: 医保报销后还能理赔吗？
A: 可以，医疗险对医保报销后的剩余部分进行赔付。

### Q3: 既往症能理赔吗？
A: 根据保险条款，既往症通常属于免责范围。

### Q4: 理赔后还能继续投保吗？
A: 可以，医疗险通常可续保，但可能调整费率。

### Q5: 异地就医如何理赔？
A: 保留好异地就医材料，按正常流程申请理赔即可。

## 五、理赔注意事项

1. **及时报案**：出险后及时通知保险公司
2. **保存原件**：所有材料保留原件
3. **如实告知**：如实填写理赔信息
4. **配合调查**：如需调查请积极配合
5. **关注进度**：及时查询理赔进度

## 六、拒赔常见原因

1. 不属于保险责任范围
2. 就诊医院不符合要求
3. 材料不完整或虚假
4. 等待期内出险
5. 既往症或免责疾病
6. 超过理赔时效
