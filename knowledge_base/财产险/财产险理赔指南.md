# 财产险理赔指南

## 一、财产险类型

### 家庭财产险
- 房屋及附属设备
- 家具、家电
- 衣物、床上用品
- 首饰、艺术品（需单独投保）

### 企业财产险
- 厂房、仓库
- 机器设备
- 原材料、成品
- 办公设备

## 二、理赔流程

### 第一步：报案

#### 报案时限
- 火灾、盗窃：立即报案
- 水灾、自然灾害：24小时内
- 其他损失：48小时内

#### 报案方式
1. 拨打保险公司客服电话
2. 通过APP在线报案
3. 到保险公司柜面报案

#### 报案内容
- 保单号码
- 损失发生时间、地点
- 损失原因
- 损失情况描述
- 联系方式

### 第二步：保护现场

#### 注意事项
1. 不要擅自清理现场
2. 拍照或录像留证
3. 采取必要措施防止损失扩大
4. 保留受损物品

#### 特殊情况
- 火灾：等待消防部门调查
- 盗窃：向公安机关报案
- 水灾：可进行必要排水

### 第三步：准备材料

#### 基础材料
1. 理赔申请书
2. 保单原件
3. 被保险人身份证
4. 银行卡信息

#### 事故证明
1. 火灾事故认定书
2. 盗窃案件报案回执
3. 气象证明（自然灾害）
4. 其他相关证明

#### 损失证明
1. 财产清单
2. 购买发票或收据
3. 财产照片
4. 评估报告（贵重物品）

#### 损失金额证明
1. 维修报价单
2. 重置发票
3. 评估报告

### 第四步：查勘定损

#### 查勘内容
- 核实事故原因
- 清点受损财产
- 评估损失程度
- 确定损失金额

#### 配合事项
1. 提供财产清单
2. 协助查勘人员工作
3. 提供相关证明材料
4. 确认定损结果

### 第五步：理赔支付

#### 支付方式
- 银行转账
- 支票支付

#### 支付时效
- 材料齐全后5-10个工作日

## 三、理赔金额确定

### 定损原则
1. **修复为主**：能修复的按修复费用赔偿
2. **重置为辅**：无法修复的按重置价值赔偿
3. **折旧扣除**：按使用年限扣除折旧

### 计算公式
```
理赔金额 = 损失金额 × 保险金额/保险价值 - 免赔额
```

### 示例
- 保险金额：100000元
- 保险价值：200000元
- 损失金额：50000元
- 免赔额：500元

理赔金额 = 50000 × (100000/200000) - 500 = 24500元

## 四、常见问题

### Q1: 部分损失如何理赔？
A: 按实际损失比例赔偿，不超过保险金额。

### Q2: 不足额投保如何处理？
A: 按保险金额与保险价值的比例赔偿。

### Q3: 贵重物品如何理赔？
A: 需提供购买凭证或评估报告，按约定价值赔偿。

### Q4: 盗窃理赔需要什么材料？
A: 需公安机关报案回执和案件未破案证明。

### Q5: 自然灾害理赔有特殊要求吗？
A: 需提供气象部门出具的灾害证明。

## 五、拒赔常见原因

1. 不属于保险责任
2. 未及时报案
3. 故意制造事故
4. 材料虚假
5. 保险标的未如实告知
6. 损失扩大未采取措施

## 六、理赔注意事项

1. **及时报案**：出险后第一时间报案
2. **保护现场**：不要擅自清理
3. **保存证据**：拍照、录像留证
4. **保留票据**：保留购买凭证
5. **配合查勘**：积极配合查勘工作
6. **核对金额**：确认理赔金额无误
